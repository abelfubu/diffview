# Slice 2: Track and highlight the Current File in the sidebar

## Summary

Implemented Current File tracking and active row highlighting in the main diff TUI sidebar.

## Changes

### Files modified

- `cli/src/cli.tsx`
  - Added `currentFileIndex` state (default: 0)
  - Added interval-based effect to compute Current File using the top-edge rule: finds the file whose header `y` position is closest to the viewport top
  - Updated `handleFileSelect` to sync `currentFileIndex` when File Picker jumps to a file
  - Passed `activeFileIndex={currentFileIndex}` to `DirectoryTreeView`

- `cli/src/components/directory-tree-view.tsx`
  - Added `activeFileIndex` prop to `DirectoryTreeViewProps`
  - Added `isActive` prop to `TreeNodeLineProps`
  - Applied additive active row highlight using `theme.primary.brighten(0.3)` as background color
  - Active highlight takes precedence over hover highlight but preserves status colors and diff stats
  - Directory rows also accept `isActive` (for future use)

- `cli/src/directory-tree.test.tsx`
  - Fixed pre-existing ellipsis character test (`"..."` → `"…"`)
  - Converted brittle inline snapshots to external snapshots (`toMatchSnapshot()`)
  - Added new test: "should highlight the active file row"

### New files

- `.changeset/sidebar-navigator-current-file.md`
- `cli/src/__snapshots__/directory-tree.test.tsx.snap` (9 snapshots generated by bun)
